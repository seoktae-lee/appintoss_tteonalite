# 떠나라이트(TteonaLite) 요구사항 명세서 (v0.2)

> 앱인토스 미니앱 · 떠나 앱의 경량 버전 — 오늘 하루를 여행처럼 기록하고, 친구와 실시간으로 공유하는 앱
>
> 작성 시작: 2026-06-21 · 최종 수정: 2026-06-21 · 상태: **초안(확장)**

---

## 1. 서비스 개요

### 1.1 한 줄 정의
오늘 방문한 장소를 사진·메모와 함께 코스로 기록하고, 그룹 친구들과 실시간 위치를 공유하는 토스 미니앱.

### 1.2 기존 떠나 앱과의 차이
- **포함**: 장소 기록(나의 오늘), 그룹 피드, 실시간 위치 공유, 웹 지도 코스 시각화
- **제외**: 영상 클립 촬영, 브이로그 자동 편집, AR/내비 코스 떠나기
- **이유**: 미니앱 환경은 설치 없이 가볍게 진입하는 것이 강점이므로, 무거운 영상·AR 기능은 본 앱(떠나)으로 유도하고 라이트는 "기록·공유"에 집중한다.

### 1.3 핵심 가치
1. **간편한 기록** — 사진 한 장 + 위치 한 번으로 오늘의 코스 완성
2. **실시간 공유** — 그룹 멤버와 지금 어디에 있는지 바로 확인
3. **수집의 재미** — 코스가 쌓이면서 나만의 여행 지도 완성

### 1.4 타깃 사용자
- 여행/나들이 기록을 부담 없이 남기고 싶은 토스 사용자
- 친구·연인·가족 단위로 함께 이동하며 위치를 공유하고 싶은 소규모 그룹
- 떠나 본 앱을 설치하기 전에 가볍게 경험해보고 싶은 잠재 고객

### 1.5 비목표 (Non-Goals)
- 장소 추천/검색 엔진, 예약·결제 연동
- 영상 편집, AR 코스, 턴바이턴 내비게이션
- 외부 SNS 공개 피드(초기 범위는 그룹 단위 비공개 공유)

## 2. 핵심 기능

### A. 나의 오늘 (코스 기록)
- 장소 추가: GPS 위치 자동 저장 + 카메라 촬영 + 메모
- 하루 단위 코스: 오늘 방문한 장소들이 시간순으로 나열
- 웹 지도(카카오맵 JS)에 핀으로 코스 시각화
- 앨범에서 사진 선택 가능
- 장소 기록 항목: 위치(위경도/주소), 사진(1장+), 메모(텍스트), 방문 시각

**세부 요구**
- 위치 권한 거부 시 수동 위치(지도 탭으로 핀 지정) 입력 폴백 제공
- 사진은 카메라/앨범 중 선택, 미첨부도 허용(위치+메모만으로 기록 가능)
- 같은 날 추가된 장소는 자동으로 같은 "오늘 코스"에 묶임(날짜 기준 그룹핑)
- 장소 기록 수정/삭제 가능

### B. 그룹 피드
- 초대 코드로 그룹 생성/참여
- 그룹 멤버의 오늘 코스를 피드 형태로 열람
- 멤버별 최신 장소 기록 표시

**세부 요구**
- 그룹 생성 시 6~8자리 영숫자 초대 코드 발급
- 한 사용자가 여러 그룹에 소속 가능, 그룹 전환 UI 제공
- 피드는 최신 기록 순 정렬, 멤버 프로필(닉네임/썸네일) 표기
- 그룹 나가기 / 그룹장 위임(또는 마지막 멤버 탈퇴 시 그룹 종료) 처리

### C. 실시간 위치 공유
- 그룹 내 멤버 위치를 웹 지도에 실시간 표시
- `startUpdateLocation` API 활용
- ON/OFF 토글 가능

**세부 요구**
- 공유 OFF가 기본값, 사용자가 명시적으로 ON 했을 때만 위치 전송(프라이버시 우선)
- 위치 갱신 주기와 백그라운드 동작은 AIT 정책 범위 내로 제한
- 마지막 갱신 시각 표시, 일정 시간 미갱신 멤버는 "오프라인"으로 표기
- 앱 종료/공유 OFF 시 위치 마커 제거

### D. 코스 보관함
- 날짜별 코스 기록 아카이브
- 코스별 장소 목록 + 사진 + 지도

**세부 요구**
- 월/일 단위 캘린더 또는 리스트로 과거 코스 탐색
- 코스 상세에서 장소 순서대로 지도 경로(폴리라인) 표시
- 코스 단위 삭제 지원

## 3. 기술 스택

- **프레임워크**: @apps-in-toss/web-framework (Granite) — 실제 설치 `2.9.2` (선언 `^2.6.1`, 정렬 권장)
- **브리지 SDK**: @apps-in-toss/web-bridge (web-framework가 re-export) — 위치·카메라·앨범·저장소·광고·결제 등 네이티브 기능
- **UI**: @toss/tds-mobile + @toss/tds-mobile-ait + Emotion
- **빌드**: React 18, Vite 6, TypeScript 5.7
- **지도**: 카카오맵 JavaScript SDK (웹뷰) — ⚠ 외부 스크립트 로딩이 AIT 웹뷰 CSP에서 허용되는지 검증 필요(§10)
- **저장소**: AIT `Storage` 브리지 (현재 코드의 `localStorage`는 임시, `Storage`로 이관 권장)
- **백엔드**: Express + JSON 파일 DB (초기) → Railway 배포
- **인증**: 토스 로그인 — 클라이언트 `appLogin()` → `{ authorizationCode, referrer }` → 서버가 mTLS로 토큰 교환
- **광고**: AIT `TossAds`(배너/전면/리워드), `GoogleAdMob`
- **결제**: AIT `IAP`(인앱결제, 구독 상품) — 떠나라이트+ 구독에 사용
- **빌드/배포 CLI**: `ait build` / `ait deploy`

## 4. 권한 (granite.config.ts)

| 권한 | access | 사용 목적 |
| --- | --- | --- |
| `geolocation` | access | 장소 기록 시 현재 위치, 실시간 위치 공유 |
| `camera` | access | 장소 사진 촬영 |
| `photos` | read | 앨범에서 사진 선택 |

- 권한은 기능 사용 시점에 요청(최초 진입 시 일괄 요청하지 않음)
- 각 권한 거부 시 대체 흐름(수동 위치 입력, 사진 미첨부) 제공

## 6. API 명세 (초안)

| 메서드 | 경로 | 설명 |
| --- | --- | --- |
| POST | `/auth/login` | 클라이언트 `appLogin()`의 `authorizationCode` 검증(mTLS 교환) → 세션 토큰 발급 |
| GET | `/courses?date=` | 내 코스 조회(날짜별) |
| POST | `/places` | 장소 기록 추가 |
| PATCH | `/places/:id` | 장소 기록 수정 |
| DELETE | `/places/:id` | 장소 기록 삭제 |
| POST | `/groups` | 그룹 생성(초대 코드 발급) |
| POST | `/groups/join` | 초대 코드로 그룹 참여 |
| GET | `/groups/:id/feed` | 그룹 피드 조회 |
| POST | `/locations` | 실시간 위치 갱신(공유 ON일 때) |
| GET | `/groups/:id/locations` | 그룹 멤버 실시간 위치 조회 |

> 클라이언트는 `src/api/client.ts`의 `api.get/post/patch/delete` 래퍼를 사용하며, `Authorization: Bearer <token>` 헤더로 인증한다.

## 7. 화면 흐름

```
[로그인] → [튜토리얼] → [홈/나의 오늘] → [장소 추가] → [코스 완성]
                              ↕
                         [그룹 피드] → [멤버 위치 지도]
                              ↕
                         [코스 보관함] → [코스 상세]
```

**화면 목록**
1. 로그인 — 토스로 시작하기
2. 튜토리얼(최초 1회) — 핵심 기능 3장 안내
3. 홈/나의 오늘 — 오늘 코스, 장소 추가 진입
4. 장소 추가 — 위치·사진·메모 입력
5. 그룹 피드 — 그룹 선택, 멤버 코스 열람
6. 멤버 위치 지도 — 실시간 위치 + 공유 토글
7. 코스 보관함 — 날짜별 아카이브
8. 코스 상세 — 장소 목록 + 지도 경로

## 8. 비기능 요구사항

- **성능**: 홈 진입 후 첫 콘텐츠 표시(FCP) 2초 이내 목표(미니앱 경량 진입)
- **프라이버시**: 위치 공유는 기본 OFF, 그룹 외부에 위치/코스 비공개
- **보안**: 토스 로그인 + mTLS, 서버는 세션 토큰으로 사용자 검증
- **안정성**: 권한·네트워크 실패에 대한 폴백 및 에러 메시지 제공
- **호환성**: 토스 인앱 웹뷰(iOS/Android) 기준 동작
- **접근성**: TDS 컴포넌트 기준 명도 대비·터치 영역 준수

## 9. 수익화 (AIT 브리지 기준)

- **배너 광고** (홈/보관함): `TossAds` `attachBanner`
- **전면 광고** (코스 저장 후): `TossAds` `loadFullScreenAd` → `showFullScreenAd`
- **(선택) 리워드 광고**: 추가 사진 슬롯/코스 잠금 해제 등 — `userEarnedReward`
- **떠나라이트+ 구독** (무제한 코스, 광고 제거): AIT `IAP` 구독 상품
  - **상품 등록**: AIT 개발자센터에 구독 상품을 SKU로 등록 → 앱에서 `IAP.getProductItemList`로 노출
  - **구매**: `IAP.createSubscriptionPurchaseOrder({ options: { sku, offerId?, processProductGrant }, onEvent, onError })`
    - `processProductGrant({ orderId, subscriptionId })`에서 **서버가 구독 지급을 기록한 뒤 `true` 반환**
  - **상태 확인**: `IAP.getSubscriptionInfo({ params: { orderId } })` → `{ status, expiresAt, isAutoRenew, gracePeriodExpiresAt, isAccessible }`
    - `isAccessible === true`일 때만 광고 제거·코스/사진 한도 해제 적용(진입 시·서버에서 재검증)
  - **주문 이력/복원**: `getCompletedOrRefundedOrders`, `getPendingOrders`로 재설치·기기 변경 시 구독 복원

## 10. AIT 브리지 API 매핑

| 기능 | AIT 브리지 API (@apps-in-toss/web-bridge 2.9.2) |
| --- | --- |
| 로그인 | `appLogin()` → `{ authorizationCode, referrer }` |
| 현재 위치(장소 기록) | `getCurrentLocation` (`getPermission`/`openPermissionDialog` 내장) |
| 실시간 위치 공유 | `startUpdateLocation(params)` → 중지 함수 반환 |
| 사진 촬영 | `openCamera` |
| 앨범 사진 선택 | `fetchAlbumPhotos` |
| 로컬 저장 | `Storage` |
| 배너 광고 | `TossAds` / `attachBanner` |
| 전면 광고 | `loadFullScreenAd` → `showFullScreenAd` |
| 리워드 광고 | `userEarnedReward` |
| 구독 상품 조회 | `IAP.getProductItemList` (콘솔에 SKU 등록 필요) |
| 구독 구매 | `IAP.createSubscriptionPurchaseOrder` (`processProductGrant`로 서버 지급) |
| 구독 상태/복원 | `IAP.getSubscriptionInfo`, `getCompletedOrRefundedOrders`, `getPendingOrders` |
| 푸시 동의 | `requestNotificationAgreement` |
| 그룹 초대(선택) | `fetchContacts` |
| 세이프에어리어 | `getSafeAreaInsets` / `subscribeSafeAreaInsets` |
| 서버 시간 | `getServerTime` |
| 리뷰 요청 | `requestReview` |

> 권한 함수(`getPermission`/`openPermissionDialog`)는 기능 사용 시점에 호출하며, `granite.config.ts`의 `permissions`와 짝을 이룬다.

## 11. 오픈 이슈 / 추후 결정

- **카카오맵 외부 SDK 로딩**: AIT 웹뷰 CSP에서 외부 `<script>` 허용 여부 검증 — 불가 시 정적 지도 이미지/타일 또는 대체 지도 검토
- **실시간 위치**: `startUpdateLocation` 갱신 주기·백그라운드/배터리 정책 (AIT 가이드 확인 필요)
- **푸시**: `requestNotificationAgreement` 도입 여부(멤버 위치 도착/피드 알림)
- JSON DB → 관리형 DB 이관 시점
- 구독 상품 SKU/가격(offerId 포함) 확정 및 개발자센터 등록
- `processProductGrant` 서버 지급 로직 + `getSubscriptionInfo` 재검증 주기 설계
- 그룹 최대 인원, 코스/사진 저장 한도(무료 vs 구독)
- 본 앱(떠나)으로의 전환 유도(딥링크) 지점
- 프레임워크 버전 정렬: `package.json ^2.6.1` ↔ 실제 `2.9.2`
