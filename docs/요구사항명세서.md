## 1. 서비스 개요

### 1.1 한 줄 정의
방문한 장소를 사진+위치+메모로 기록하면 하루 코스가 자동 생성되고, 다른 유저의 인기 코스를 탐색하고, 그룹에서 코스와 댓글을 공유하는 앱인토스 미니앱.

### 1.2 앱 이름
**떠나라이트** (영문: TteonaLite)

### 1.3 핵심 목적
- **독립적 서비스**: 라이트만으로도 완결된 코스 기록/공유 경험 제공
- **본앱 유입 통로**: "영상으로 더 멋지게 기록하고 싶다면?" → 떠나 본앱 다운로드 유도

### 1.4 해결하는 문제
- 데이트/여행/일상 코스를 간편하게 기록하고 싶다
- 다른 사람들의 인기 코스를 참고하고 싶다
- 친구/연인과 코스를 공유하고 소통하고 싶다

### 1.5 핵심 가치
1. **간편한 기록** — 사진+위치+메모만으로 하루 코스 자동 생성
2. **코스 탐색** — 인기 코스 발견 + 태그별 필터 + 좋아요/북마크
3. **그룹 공유** — 초대코드로 그룹 생성, 멤버 코스 피드 + 댓글
4. **수집의 재미** — 코스 아카이브, 캘린더 뷰

| 여행자 | 혼자/친구와 여행 중 장소 기록 | "여행 동선을 정리하고 싶다" |
| 일상 기록러 | 카페, 맛집 등 일상 탐방 기록 | "자주 가는 곳들을 모아두고 싶다" |

## 3. 핵심 기능 A — 코스 기록 ("나의 오늘")

### 3.1 기록 방식
- 장소 하나씩 추가하면 **당일 코스가 자동 생성**
- 하루에 여러 장소를 추가하면 하나의 코스로 묶임

### 3.2 장소 기록 항목

| 항목 | 필수/선택 | 입력 방식 |
|------|-----------|-----------|
| 사진 | **필수** | 카메라 촬영 또는 갤러리 선택 |
| GPS 위치 | **필수** | 자동 수집 (수동 검색 폴백) |
| 주소 | 자동 | 역지오코딩으로 자동 생성 |
| 메모 | 선택 | 텍스트 입력 |

### 3.3 코스 정보

| 항목 | 설명 |
|------|------|
| 코스 이름 | 자동 생성 ("6월 26일 코스") + 직접 수정 가능 |
| 태그 | 데이트 / 여행 / 맛집탐방 / 카페투어 / 산책 / 기타 |
| 공개 설정 | 공개 / 비공개 선택 (공개 시 탐색에 노출) |
| 장소 순서 | 방문 시간 순 자동 정렬 |

### 3.4 코스 수정/삭제
- 장소별 수정 (사진 변경, 메모 수정)
- 장소 삭제 / 코스 전체 삭제

## 4. 핵심 기능 B — 코스 탐색

### 4.1 탐색 화면
- **태그별 필터**: 데이트 / 여행 / 맛집탐방 / 카페투어 / 산책 / 전체
- **정렬**: 인기순 (좋아요 기준)
- **코스 카드**: 대표 사진 + 코스명 + 태그 + 장소 수 + 좋아요 수

### 4.2 코스 상세
- 코스 이름 + 작성자 별명 + 날짜
- 장소 리스트 (사진 + 주소 + 메모)
- **지도에 장소 핀 표시** (Leaflet + OpenStreetMap)
- 좋아요 버튼 + 북마크 버튼
- **본앱 유도**: "영상으로 더 멋지게 기록하고 싶다면? 떠나 앱 다운로드"

### 4.3 신고 기능
- 부적절한 코스 신고 (3회 누적 시 자동 숨김)

---

## 5. 핵심 기능 C — 그룹

### 5.1 그룹 생성/참여
- 6자리 초대코드로 그룹 생성/참여
- 그룹 이름 + 내 닉네임 설정

### 5.2 그룹 피드
- 멤버들의 코스를 피드 형태로 확인
- 코스에 댓글 달기

### 5.3 실시간 위치 공유
- **수동 공유 방식**: 사용자가 "내 위치 공유" 버튼을 누를 때만 위치 업데이트
- 그룹 멤버의 마지막 공유 위치를 지도에 표시
- 마지막 업데이트 시간 표시
- **v1.1에서 자동 폴링 방식 업그레이드 검토**

### 5.4 그룹 제한
- MVP: 제한 없음 (PRO 모델은 추후 결정)

## 10. 화면 흐름 (IA)

### 10.1 온보딩 (4~5단계)
1. 마스코트 소개
2. 코스 기록 안내 (사진+위치+메모)
3. 코스 탐색 안내 (인기 코스 발견)
4. 그룹 공유 안내
5. 시작하기

### 10.2 하단 탭 (4개)
```
[홈(내코스)] [탐색] [그룹] [설정]
```

### 10.3 화면 목록
1. 온보딩 (4~5슬라이드)
2. 로그인 (토스)
3. 별명 설정
4. **홈/내코스** — 오늘의 코스 + "장소 추가" CTA + 지도 미니뷰
5. **장소 추가** — 카메라/갤러리 + 위치 + 메모
6. **코스 상세** — 장소 리스트 + 지도 + 좋아요/북마크
7. **코스 편집** — 이름/태그/공개설정 변경
8. **탐색** — 태그 필터 + 인기 코스 카드
9. **그룹 목록** — 내 그룹 리스트 + 생성/참여
10. **그룹 상세** — 멤버 코스 피드 + 댓글 + 위치 공유
11. **코스 아카이브** — 캘린더 + 과거 코스
12. **설정** — 프로필, 알림, 이용약관, 본앱 유도, 탈퇴
13. **프리미엄** (추후)

### 10.4 주요 플로우
```
[홈] ──장소 추가──> [카메라/갤러리] ──> [위치 확인] ──> [메모 입력] ──> [홈(코스에 장소 추가됨)]
   │
   ├──코스 탭──> [코스 상세] ──> [지도 + 장소 리스트]
   │
   ├──탐색 탭──> [태그 필터 + 인기 코스] ──> [코스 상세] ──> [좋아요/북마크]
   │
   ├──그룹 탭──> [그룹 목록] ──> [그룹 상세 (피드+댓글+위치)]
   │
   └──설정 탭──> [프로필/알림/약관/본앱유도/탈퇴]
```

## 11. 기술 스택

### 11.1 프런트엔드
- **프레임워크**: `@apps-in-toss/web-framework`
- **UI**: `@toss/tds-mobile` + Emotion
- **지도**: Leaflet + OpenStreetMap (react-leaflet)
- **베이스**: React 18 · Vite · TypeScript

### 11.2 백엔드
- **런타임**: Node.js + Express + TypeScript (빨래요/내차여기 동일 패턴)
- **DB**: JSON 파일 기반
- **호스팅**: Railway → NHN Cloud VM (7월 이후)
- **인증**: JWT + 토스 로그인 (mTLS)

### 11.3 외부 서비스
- **역지오코딩**: Nominatim (무료)
- **지도**: Leaflet + OpenStreetMap (무료)
- **푸시**: 앱인토스 스마트 발송

## 12. 본앱 연동

### 12.1 v1 — 연동 없음
- 라이트는 독립 서버/DB로 운영
- 본앱과 데이터 공유 안 함

### 12.2 v1.1+ — 코스 열람 연동 (검토)
- 본앱 Firebase에서 인기 코스를 라이트에서 읽기 전용으로 표시
- API 하나만 추가하면 가능

### 12.3 본앱 유도
- 코스 상세에서 "영상으로 더 멋지게 기록하고 싶다면? 떠나 앱 다운로드" 배너
