# 떠나라이트(TteonaLite) 요구사항 명세서 (v0.1 초안)

> 앱인토스 미니앱 · 떠나 앱의 경량 버전 — 오늘 하루를 여행처럼 기록하고, 친구와 실시간으로 공유하는 앱
>
> 작성 시작: 2026-06-21 · 상태: **초안**

---

## 1. 서비스 개요

### 1.1 한 줄 정의
오늘 방문한 장소를 사진·메모와 함께 코스로 기록하고, 그룹 친구들과 실시간 위치를 공유하는 토스 미니앱.

### 1.2 기존 떠나 앱과의 차이
- **포함**: 장소 기록(나의 오늘), 그룹 피드, 실시간 위치 공유, 웹 지도 코스 시각화
- **제외**: 영상 클립 촬영, 브이로그 자동 편집, AR/내비 코스 떠나기

### 1.3 핵심 가치
1. **간편한 기록** — 사진 한 장 + 위치 한 번으로 오늘의 코스 완성
2. **실시간 공유** — 그룹 멤버와 지금 어디에 있는지 바로 확인
3. **수집의 재미** — 코스가 쌓이면서 나만의 여행 지도 완성

---

## 2. 핵심 기능

### A. 나의 오늘 (코스 기록)
- 장소 추가: GPS 위치 자동 저장 + 카메라 촬영 + 메모
- 하루 단위 코스: 오늘 방문한 장소들이 시간순으로 나열
- 웹 지도(카카오맵 JS)에 핀으로 코스 시각화
- 앨범에서 사진 선택 가능

### B. 그룹 피드
- 초대 코드로 그룹 생성/참여
- 그룹 멤버의 오늘 코스를 피드 형태로 열람
- 멤버별 최신 장소 기록 표시

### C. 실시간 위치 공유
- 그룹 내 멤버 위치를 웹 지도에 실시간 표시
- startUpdateLocation API 활용
- ON/OFF 토글 가능

### D. 코스 보관함
- 날짜별 코스 기록 아카이브
- 코스별 장소 목록 + 사진 + 지도

## 3. 기술 스택

- **프레임워크**: @apps-in-toss/web-framework (Granite)
- **UI**: @toss/tds-mobile + Emotion
- **빌드**: React 18, Vite 6, TypeScript 5.6
- **지도**: 카카오맵 JavaScript SDK (웹)
- **백엔드**: Express + JSON DB (→ Railway 배포)
- **인증**: 토스 로그인 (appLogin + mTLS)

---

## 4. 권한
- `geolocation` (access): 장소 기록 + 실시간 위치
- `camera` (access): 장소 사진 촬영
- `photos` (read): 앨범에서 사진 선택
