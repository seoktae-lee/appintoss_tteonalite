## 1. 서비스 개요

### 1.1 한 줄 정의
방문한 장소를 사진+위치+메모로 기록하면 하루 코스가 자동 생성되고, 다른 유저의 인기 코스를 탐색하고, 그룹에서 코스와 댓글을 공유하는 앱인토스 미니앱.

### 1.2 앱 이름
**떠나라이트** (영문: TteonaLite)

### 1.3 핵심 목적
- **독립적 서비스**: 라이트만으로도 완결된 코스 기록/공유 경험 제공
- **본앱 유입 통로**: "영상으로 더 멋지게 기록하고 싶다면?" → 떠나 본앱 다운로드 유도

### 1.4 해결하는 문제
- 데이트/여행/일상 코스를 간편하게 기록하고 싶다
- 다른 사람들의 인기 코스를 참고하고 싶다
- 친구/연인과 코스를 공유하고 소통하고 싶다

### 1.5 핵심 가치
1. **간편한 기록** — 사진+위치+메모만으로 하루 코스 자동 생성
2. **코스 탐색** — 인기 코스 발견 + 태그별 필터 + 좋아요/북마크
3. **그룹 공유** — 초대코드로 그룹 생성, 멤버 코스 피드 + 댓글
4. **수집의 재미** — 코스 아카이브, 캘린더 뷰

## 3. 핵심 기능 A — 코스 기록 ("나의 오늘")

### 3.1 기록 방식
- 장소 하나씩 추가하면 **당일 코스가 자동 생성**
- 하루에 여러 장소를 추가하면 하나의 코스로 묶임

### 3.2 장소 기록 항목

| 항목 | 필수/선택 | 입력 방식 |
|------|-----------|-----------|
| 사진 | **필수** | 카메라 촬영 또는 갤러리 선택 |
| GPS 위치 | **필수** | 자동 수집 (수동 검색 폴백) |
| 주소 | 자동 | 역지오코딩으로 자동 생성 |
| 메모 | 선택 | 텍스트 입력 |

### 3.3 코스 정보

| 항목 | 설명 |
|------|------|
| 코스 이름 | 자동 생성 ("6월 26일 코스") + 직접 수정 가능 |
| 태그 | 데이트 / 여행 / 맛집탐방 / 카페투어 / 산책 / 기타 |
| 공개 설정 | 공개 / 비공개 선택 (공개 시 탐색에 노출) |
| 장소 순서 | 방문 시간 순 자동 정렬 |

### 3.4 코스 수정/삭제
- 장소별 수정 (사진 변경, 메모 수정)
- 장소 삭제 / 코스 전체 삭제

## 4. 핵심 기능 B — 코스 탐색

### 4.1 탐색 화면
- **태그별 필터**: 데이트 / 여행 / 맛집탐방 / 카페투어 / 산책 / 전체
- **정렬**: 인기순 (좋아요 기준)
- **코스 카드**: 대표 사진 + 코스명 + 태그 + 장소 수 + 좋아요 수

### 4.2 코스 상세
- 코스 이름 + 작성자 별명 + 날짜
- 장소 리스트 (사진 + 주소 + 메모)
- **지도에 장소 핀 표시** (Leaflet + OpenStreetMap)
- 좋아요 버튼 + 북마크 버튼
- **본앱 유도**: "영상으로 더 멋지게 기록하고 싶다면? 떠나 앱 다운로드"

### 4.3 신고 기능
- 부적절한 코스 신고 (3회 누적 시 자동 숨김)

## 10. 화면 흐름 (IA)

### 10.1 온보딩 (4~5단계)
1. 마스코트 소개
2. 코스 기록 안내 (사진+위치+메모)
3. 코스 탐색 안내 (인기 코스 발견)
4. 그룹 공유 안내
5. 시작하기

### 10.2 하단 탭 (3개)
```
[홈(지도)] [탐색] [설정]
```

### 10.3 화면 목록
1. 온보딩 (3슬라이드)
2. 로그인 (토스)
3. 별명 설정
4. **홈** — 전체화면 지도 + 코스 마커 + "나의 오늘" CTA
5. **나의 오늘 기록 모드** — 지도 + "여기서 촬영" + 장소 카운트
6. **장소 선택** — 주변 POI 리스트 (역지오코딩)
7. **카메라** — 사진 촬영/갤러리
8. **기록 완료** — 번호 마커 + 장소 칩 + 여기서 촬영/오늘 종료
9. **오늘 종료** — 코스 저장 시트
10. **코스 상세** — 지도+경로+사진+장소목록+좋아요/북마크+"이 코스로 떠나기"
11. **코스 진행 중** — 경로 네비게이션 + 거리 안내
12. **코스 편집** — 순서 변경 + 건너뛰기
13. **탐색** — 태그 필터 + 인기 코스 카드
14. **코스 아카이브** — 캘린더 + 과거 코스
15. **설정** — 프로필, 알림, 이용약관, 본앱 유도, 탈퇴
16. **프리미엄** (추후)

### 10.4 주요 플로우 요약
```
[홈(지도)] ──나의 오늘──> [기록 모드] ──여기서 촬영──> [사진 촬영] ──> [메모 입력] ──> [장소 추가(반복)]
   │                                                                              ──오늘 종료──> [코스 저장]
   │
   ├──코스 마커 탭──> [코스 미리보기] ──> [코스 상세] ──이 코스로 떠나기──> [코스 진행 중]
   │
   ├──탐색 탭──> [태그 필터 + 인기 코스] ──> [코스 상세] ──> [좋아요/북마크]
   │
   └──설정 탭──> [프로필/알림/약관/아카이브/본앱유도/탈퇴]
```

### 10.5 "나의 오늘" 상세 플로우 (본앱 참고)

#### Step 1: 홈 → 기록 시작
- 홈 지도 하단의 **"나의 오늘"** 버튼 탭
- 기록 모드 진입 → 지도가 현재 위치 줌인
- 상단에 **"● 나의 오늘 기록 중"** 상태 표시 + **"0곳"** 카운트 뱃지

#### Step 2: 여기서 촬영
- 하단 **"여기서 촬영"** 버튼 탭
- 바로 **카메라/갤러리 선택** 열림 (네이티브 파일 피커)
- 사진 선택 후 → **메모 입력 바텀시트** 표시
  - 사진 미리보기
  - 메모 입력 (선택)
  - "취소" / "장소 추가" 버튼
- 장소 추가 시 → GPS 위치 자동 수집 + 역지오코딩 주소 변환
- 지도에 **번호 마커** 추가 (①, ②, ③...)
- 하단에 **장소 칩** 표시 (가로 스크롤)

#### Step 3: 반복 기록
- "여기서 촬영" 반복하여 여러 장소 추가
- 장소마다 번호 마커가 지도에 누적

#### Step 4: 오늘 종료
- 장소가 1개 이상일 때 **"오늘 종료"** 버튼 활성화
- 탭 시 **TDS ConfirmDialog** 표시: "오늘을 마칠까요?"
- 확인 → 코스로 자동 저장 (제목: "6월 27일 코스", 태그/공개설정 가능)
- 홈으로 복귀

#### Step 5: 이어하기 (앱 재진입 시)
- 기록 중인 세션이 있으면 홈에 **"이어하기"** 버튼 표시
- 탭 시 **바텀시트** 표시:
  - "오늘 기록이 남아있어요" + 장소 수 + 시작 시간
  - **"이어서 기록하기"** → 기록 모드로 복귀 (기존 장소 유지)
  - **"새로 시작하기"** → 기존 세션 폐기 + 홈으로 복귀 (이어하기 사라짐)

#### Step 6: 나루 로딩
- 세션 시작/종료/장소 업로드 등 대기 시간에 **나루(점프 포즈) 로딩 인디케이터** 표시

---

### 10.6 "이 코스로 떠나기" 상세 플로우 (본앱 참고)

#### Step 1: 코스 발견
- **탐색 탭**에서 태그별 인기 코스 카드 탐색
- 또는 **홈 지도**에서 코스 마커 탭 → 코스 미리보기 바텀시트

#### Step 2: 코스 상세 확인
- 코스 상세 페이지 진입
  - 상단: 지도에 **번호 마커 + 경로 점선(polyline)** 표시
  - 작성자 닉네임 + 날짜
  - 사진 스와이프 (dot indicator)
  - 태그 + 좋아요 수
  - **장소 목록**: 번호 + 사진 + 장소명 + 주소 + 메모
  - **좋아요 / 북마크** 버튼
  - **신고** 버튼 (TDS ConfirmDialog)
  - 본앱 유도 배너: "영상으로 더 멋지게 기록하고 싶다면? 떠나 앱 다운로드"

#### Step 3: 이 코스로 떠나기
- 하단 고정 CTA **"이 코스로 떠나기"** 버튼 탭
- 기록 모드 시작 (나의 오늘과 동일한 기록 플로우)
- 코스의 장소들을 참고하며 직접 방문 + 촬영

#### Step 4: 코스 완성
- 방문한 장소들로 나만의 새 코스가 자동 생성
- "오늘 종료" → 코스 저장

## 11. 기술 스택

### 11.1 프런트엔드
- **프레임워크**: `@apps-in-toss/web-framework`
- **UI**: `@toss/tds-mobile` + Emotion
- **지도**: Leaflet + OpenStreetMap (react-leaflet)
- **베이스**: React 18 · Vite · TypeScript

### 11.2 백엔드
- **런타임**: Node.js + Express + TypeScript (빨래요/내차여기 동일 패턴)
- **DB**: JSON 파일 기반
- **호스팅**: Railway → NHN Cloud VM (7월 이후)
- **인증**: JWT + 토스 로그인 (mTLS)

### 11.3 외부 서비스
- **역지오코딩**: Nominatim (무료)
- **지도**: Leaflet + OpenStreetMap (무료)
- **푸시**: 앱인토스 스마트 발송
